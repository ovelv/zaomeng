# 造梦工厂 (Dream Factory)

造梦工厂是一个基于 Next.js 开发的 AI 多模态视频生成平台，专注于将创意转化为高质量的视频内容。全面接入火山引擎（Volcengine）的大模型能力，实现从文案分镜到图像生成，再到视频渲染的全链路自动化。

## 核心功能

### 1. 萌宠视频生成 (AI Pet Video)
- **智能分镜**：输入故事主题，调用大语言模型自动生成详细的分镜脚本和运镜提示词。
- **一致性生图**：结合全局角色设定，使用 Seedream 生成各分镜的角色底图，保证长相和画风的一致性。
- **多镜头叙事**：串联 Seedance 2.0 图生视频能力，配合各分镜的运镜提示词，完成连贯的多镜头叙事视频。
- **精细化控制**：支持对单个分镜的图像或视频进行单独重新生成（Retry），拒绝“一子落错满盘皆输”，极大节省时间和成本。

### 2. 口播视频生成 (Oral Video)
- **智能长文案切分**：输入长篇口播文案，自动按字数和语义停顿切分为短视频脚本。
- **固定人设**：每段使用相同的 `assetId`、人设模板和镜头模板，保持口播人物形象绝对稳定。
- **时长自适应**：每段视频时长可单独控制（支持 4-15 秒）。

### 3. 自定义视频生成 (Custom Video)
- 提供高级的视频生成沙盒，直接调用 Seedance 2.0。
- 支持自由配置：提示词、首尾帧参考图、视频比例、分辨率和视频时长，满足专业创作者的定制化需求。

### 4. 任务历史与工作流
- **历史记录**：所有生成记录保存在浏览器 LocalStorage 中，随时回顾。
- **一键复用**：在历史记录中可以一键带入之前的配置参数，快速开始新的生成。
- **无水印与带音频**：底层 API 深度优化，生成的图像和视频均去除了官方的水印，并支持开启 TTS 音频合成功能。

## 快速启动

造梦工厂完全解耦了服务端的环境变量配置，**所有 API 密钥和模型参数均通过前端页面进行配置**，开箱即用。

1. 安装依赖

```bash
npm install
```

2. 启动开发环境

```bash
npm run dev
```

3. 访问应用与配置
打开 [http://localhost:3000](http://localhost:3000)
- 点击右上角的 **设置 (Settings)** 按钮。
- 填入你的火山引擎配置（保存在本地浏览器中，安全便捷）：
  - `ARK_API_KEY`: 你的方舟 API Key
  - `ARK_BASE_URL`: 默认为 `https://ark.cn-beijing.volces.com/api/v3`
  - `STORYBOARD_MODEL`: 分镜大语言模型接入点 (如 `ep-xxx`)
  - `SEEDREAM_MODEL`: Seedream 图像模型接入点 (如 `ep-xxx`)
  - `SEEDANCE_MODEL`: Seedance 视频模型接入点 (如 `ep-xxx`)

## 技术栈

- **框架**: Next.js 14+ (App Router)
- **样式**: Tailwind CSS (支持毛玻璃质感、动画效果与自适应布局)
- **状态管理**: Zustand (前端任务队列与状态机)
- **图标与资产**: Lucide React, Next.js `@vercel/og` 动态生成 Favicon
- **AI 驱动**: 火山引擎 (Volcengine) Doubao 系列模型

## 架构特点

- **零服务端状态**：API Key 等敏感信息和生成进度完全由客户端管理并随请求传递，服务端 API 为纯无状态函数。
- **局部重试机制**：复杂的图生视频 Pipeline 支持断点重跑，极大节省了生成时间和 Token 成本。
- **无感等待体验**：现代化的 SaaS 交互设计，后台静默轮询任务状态。
